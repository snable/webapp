# This is a template webapp

It have gulp config with tasks
* stylus
* jade
* browserify (coffee)
* bower
* images (simple copy)
* sprites

## Usage

After clonning this - run:

```
npm install
bower install
gulp
```

and go to

`
localhost:8080
`

in your browser
